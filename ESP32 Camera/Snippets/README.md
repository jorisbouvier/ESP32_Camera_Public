# Snippets

This folder contains code snippets, all the code snippets are just here to help you to understand how to use the ESP32-CAM module.
Each snippet is a simple example that demonstrates a specific feature of the ESP32-CAM module.

## Snippets

- [ESPMQTT]
    This snippet demonstrates how to send messages using the MQTT protocol.

- [CameraWebServer]
    This snippet demonstrates how to take a picture or live stream using the ESP32-CAM module.

- [SCW_Photobooth]
    This snippet demonstrates how to create a photobooth using the ESP32-CAM module and MQTT communication.
